当前代码功能：
1. 地图保存
2. 加载地图后进行重定位

代码编译：
build.sh　	----- 对相关的第三方库进行编译的脚本；注意该代码依赖于opencv，请提前安装
build_ros.sh 	----- 讲ORB-SLAM编译ROS为ROS包;

代码运行：
1. 地图保存
	----- 编译完成后；
	----- 运行 run.sh　脚本, 根据提示选择不加载地图
	----- 运行 play.sh 脚本，播放已经录制好的数据包（bag目录下的数据包）
	----- 待 play.sh　脚本播放完毕之后，按 ctrl + c　终止run.sh脚本，根据提示选择：！！！保存地图
	----- 地图保存完成！
2. 加载地图后进行重定位
	----- 编译完成后；
	----- 运行 run.sh　脚本, 根据提示选择加载地图
	----- 播放 数据包 或则 接入真实的
	----- 待完成重定位之后，按 ctrl + c　终止run.sh脚本，根据提示选择：！！！不保存地图
	----- 重定位完成
